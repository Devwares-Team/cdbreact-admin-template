# Contrast React Admin Dashboard (Free)

This project was with the CDBReact Package [CDBReact](https://github.com/Devwares-Team/cdbreact).

## Install

```bash
npm install
```

or 

```bash
yarn install
```

## Usage

```bash
npm start
```
or

```bash
yarn start
```
